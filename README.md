## Usage
### Batch file

Run the batch file named `run` to run the program.
```
$ run.bat
```

### Compile
Compile the program using `g++` and run the program.
```
$ g++ src/* -o main.exe && main
```

## Location of Main Function
```
src/Main.cpp
```
